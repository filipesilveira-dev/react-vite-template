# 🚀 React + TypeScript + Tailwind CSS Template

Bem-vindo! Este é um template base (boilerplate) minimalista e ultra-rápido para iniciar novos projetos React sem perder tempo limpando arquivos de demonstração, logos ou contadores. 

Criado com o objetivo de ir direto ao código autoral, ele já vem configurado com as tecnologias mais modernas do ecossistema frontend.

## 🛠️ O que tem na caixa?

*   ⚡ **Vite** – O build tool mais rápido do mercado.
*   ⚛️ **React 19 + React Compiler** – Performance otimizada automaticamente, sem necessidade de encher o código com `useMemo` ou `useCallback`.
*   📘 **TypeScript** – Tipagem estática para um código mais seguro e auto-explicativo.
*   🎨 **Tailwind CSS v4** – Estilização utilitária moderna de alta performance, integrada direto no Vite (sem arquivos de configuração poluídos).
*   📁 **Estrutura de Pastas Pronta** – Diretórios pré-organizados para `components`, `hooks`, `types` e `assets`.

---

## 🚀 Como usar em 3 segundos (Sem clonar o histórico do Git)

Para criar um novo projeto utilizando este template como base, você não precisa dar `git clone`. Use o **`degit`**, que baixa apenas os arquivos limpos e prontos para o uso.

No seu terminal, rode o comando substituindo pelos seus dados:

```bash
npx degit SEU_USUARIO_DO_GITHUB/NOME_DESTE_REPOSITORIO meu-novo-projeto
